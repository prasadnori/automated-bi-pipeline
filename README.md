# automated-bi-pipeline

An object-oriented pipeline for automating exploratory data analysis (EDA) in Business Intelligence workflows.
The system reads datasets, generates frequency tables, and automatically produces visualizations such as bar charts and pie charts.

# pipeline workflow
Dataset
   ↓
Data Reader
   ↓
Frequency Table Generator
   ↓
Visualization Engine
   ↓
Charts & Analytical Outputs


# project structure

automated-bi-pipeline
